GNU/Linux auto installation files.

## Install

Automatic installation is done by setting a command in installation menu.

#### CentOS / Fedora

```
inst.ks=https://uaik.github.io/os.id.txt
```

#### Debian / Ubuntu

```
url=https://uaik.github.io/os.id.txt
```
